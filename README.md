Database Validation Automation (SQL + Java)

Automated tests validating backend database state after API or UI actions.

## Tech Stack
- Java
- JDBC
- SQL (PostgreSQL or MySQL)
- TestNG

## What This Project Covers
- Database connection using JDBC
- Validation of inserts, updates, and deletes
- Comparison of API responses with database records
- Transaction handling and rollbacks
